How to publish a new article

Option A — Use this Dashboard (Easiest)

  1. Fill in the form on the Editor tab (title, date, description, tags, image, content).
  2. Click Generate Article Files.
  3. Download your-article.html and the updated articles.json.
  4. Upload both files to your GitHub repo (drag & drop in the web UI).
  5. GitHub Pages redeploys automatically — your article is live in ~60 seconds!

Option B — JSON Draft + Python Script

  1. Create a JSON file in drafts/ folder (see JSON tab for format).
  2. Run python generate_seo.py locally.
  3. Commit all generated files to GitHub.
  4. Or use the GitHub Actions workflow (workflows/seo.yml) to auto-publish on push.

Option C — Direct HTML Edit

  1. Copy article_template.html and rename it.
  2. Replace all {{PLACEHOLDER}} values.
  3. Add the article to articles.json manually.
  4. Upload to GitHub.
📌 Uploading to GitHub (web UI)
Go to your repo → click Add file → Upload files → drag your files → click Commit changes. GitHub Pages will rebuild automatically.
